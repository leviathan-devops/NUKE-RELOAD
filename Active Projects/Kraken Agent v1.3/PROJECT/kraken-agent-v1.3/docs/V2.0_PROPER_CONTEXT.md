```

    ↓

    ↓
spawn_cluster_task tool called
    ↓
AsyncDelegationEngine.delegate() queues task
    ↓
ClusterScheduler.assignCluster() selects cluster
    ↓
ClusterInstance.executeTaskAsync() marks agent busy
    ↓
ClusterInstance.executeOnAgent()  ← REAL EXECUTION
    ↓
spawn('python3', ['opencode_agent.py', ...args])
    ↓
Docker spawns container with opencode image
    ↓
Container runs with shark-agent or manta-agent mounted
    ↓
Real agent executes real task in isolated container
    ↓
Result returned via JSON
    ↓
Kraken stores patterns in Hive (if success)
    ↓
Real result to user
```
